# Gestão-Produtos
Projeto para aprofundar conhecimentos em OOP e Persistência de dados em Java

## Requisitos:
### Pessoa
- Admin
- - CRUD Produtos
  - CRUD Pessoas

- Pessoa
- - Vendas em Produtos

### Produtos
- CRUD
- - Filtrar preço ou categoria
- Relatórios
- - Mostrar produtos que mais saem ou produtos que estão acabando
- Logs
- - Mostrar qual usuário vendou o que
